# Bilibili App 接口模拟 - v8.62.0

本项目逆向分析了哔哩哔哩（Bilibili）安卓客户端 v8.62.0 版本，并使用 Python 实现了与播放量上报相关的核心接口的模拟访问。

## 工作原理

本脚本的核心是通过**模拟生成**一个完整的、一致的安卓设备指纹来与Bilibili服务器进行交互，该指纹包括但不限于：

*   `buvid` (设备ID)
*   `oaid` (开放匿名设备标识符)
*   `android_id`
*   设备型号、系统版本等静态参数

**所有这些设备信息均为随机生成或硬编码的伪造数据，不涉及也绝不会读取或暴露您真实的设备信息。**
